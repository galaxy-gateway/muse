## Methodology

Token counts are calculated by reading the actual source files an AI agent would
need to explore for each task, using the ~4 chars/token heuristic (standard for
GPT/Claude tokenizers). Tool call overhead is estimated at ~75 tokens per call
(command text + result formatting). The "with codesight" count uses the real
CODESIGHT.md output size, proportioned to the sections relevant to each task.
